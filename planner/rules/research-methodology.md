# 리서치 방법론 규칙

체계적이고 신뢰할 수 있는 리서치를 위한 방법론.

## 1. 출처 신뢰도 [CRITICAL]

모든 데이터에 출처와 신뢰도 표시.

```markdown
BAD:
"AI 시장은 2025년까지 $500B 규모"

GOOD:
"AI 시장은 2025년 $407B 규모 예상 (IDC, 2024.06, 신뢰도: HIGH)"
```

### 신뢰도 등급
| 등급 | 기준 |
|------|------|
| HIGH | 1차 소스, 공신력 있는 기관 (Gartner, IDC, 정부 통계) |
| MEDIUM | 2차 소스, 업계 보고서, 전문 미디어 |
| LOW | 블로그, 소셜 미디어, 미확인 소스 |

## 2. 교차 검증 [HIGH]

핵심 데이터는 2개 이상 소스에서 교차 검증.

```markdown
BAD:
단일 소스의 수치를 그대로 인용

GOOD:
"시장 규모: $12B (Gartner) / $11.5B (IDC) / $13.2B (Statista)"
→ 합의 범위: $11.5-13.2B
```

## 3. 편향 인식 [HIGH]

리서치 결과에 잠재적 편향 명시.

```markdown
## 한계 및 편향
- 영어권 소스 위주 (아시아 시장 데이터 부족)
- 2023년 이전 데이터 포함 (급변 시장에서 신뢰도 감소)
- 생존자 편향: 성공 사례 위주 분석
```

## 4. 결론 확신도 [MEDIUM]

```markdown
## 결론
[결론 내용] — 확신도: HIGH/MEDIUM/LOW

확신도 근거:
- 3개 독립 소스 교차 확인 (HIGH)
- 최근 6개월 내 데이터 (HIGH)
- 단일 소스, 추정치 포함 (LOW)
```
